# Library Management System

The Library Management System is a web-based application that allows users to manage and track books in a library. It provides functionalities to add, update, and delete library records, as well as view the existing records.

## Features

- Add new books to the library with details such as book name, ID, author name, and status.
- Update existing book records, including book name, author name, status, and issuer ID (if the book is issued).
- Delete book records from the library.
- View the complete list of books in the library with their details.
- User-friendly interface for easy interaction and navigation.

## Technologies Used

- HTML
- CSS
- Python
- Flask (Python web framework)
- SQLite (Database)

## Getting Started

1. Clone the repository:
```

git clone https://github.com/your-username/library-management-system.git

```

2. Navigate to the project directory:

```

cd library-management-system

```

3. Install the dependencies:

```

pip install -r requirements.txt

```

4. Start the application:

```

python app.py

```

5. Open your web browser and visit `http://localhost:5000` to access the Library Management System.

## Usage

- **Add a Book**: Fill in the book details in the provided form and click on "Add Record" to add a new book to the library.
- **Update a Book**: Click on the "Update" link next to a book record, make the necessary changes, and click on "Update Record" to save the changes.
- **Delete a Book**: Click on the "Delete" link next to a book record to remove it from the library.
- **View Library**: Click on the "View Library" link to see the complete list of books in the library.
